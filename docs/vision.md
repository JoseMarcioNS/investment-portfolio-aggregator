# Vision — Investment Portfolio Aggregator

## Problem

Investors often have accounts in multiple brokers and platforms.
Tracking the full investment portfolio becomes difficult.

Users need a way to consolidate:

* stocks
* crypto
* ETFs
* funds

from multiple brokers into a single dashboard.

---

## Goal

Create a backend system capable of:

* consolidating investment data from multiple brokers
* calculating portfolio positions
* calculating portfolio value
* tracking historical portfolio performance

---

## Core Features

1. Connect broker accounts
2. Import investment transactions
3. Calculate current portfolio positions
4. Display portfolio summary
5. Track historical portfolio value

---

## Non-Functional Requirements

The system must:

* support asynchronous processing
* support scaling
* support event driven architecture
* keep historical data for auditing
